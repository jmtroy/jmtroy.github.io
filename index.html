About Christmas Card Trivia

Christmas Card Trivia - a game for the entire family

Since I was a child Christmas Eve was when my family gathered together to share a meal, exchange gifts and very often to play games. In 2007 I made a game that was suprising, fun, beautiful and also celebrated the birth of Jesus Christ. The game is a trivia game about christmas. Each question was printed on a single sheet of paper and folded like a christmas card, and each answer was also printed on its own christmas card.

Now the game is freely available for everyone to download and print the christmas cards, place them around the home, a let their family enjoy a unique game of trivia.

Printing The Game

How to print the game

This game has 70 question cards and 70 answer cards that will need to be printed. After you download the game (see below) 140 pdf files will be found in the christmascard_pdfs folder.

Each card is one pdf file to be printed on both sides of 8.5 x 11 inch paper.

  • Download the game using the button below.
  • Unzip the file.
  • Find the 140 pdfs in the unzipped christmascard_pdfs folder.
  • Each pdf file is printed on both sides of 8.5 x 11 inch paper.
  • The paper can be normal weight, but a heavier weight paper is better.
  • A print shop (Office Max or Staples) can also print the cards, and can provide paper that is scored for easy folding.
  • Each printed pdf is folded to make a christmas card.

Playing The Game

Playing the Christmas Card Trivia Game

Objective: To collect as many matching Question and Answer Cards as you can.

A hint about matching: For a matching pair of cards, not only will the question and answer match, but the border pattern and the picture on the inside will match as well.

Setup: The Christmas Card Trivia Game is meant to be set up and played throughout the entire house. Fold the cards and randomly set them on any suitable surface; tables, kitchen counters, bedroom dressers, the dining room buffet, anywhere!

Teams: The Christmas Card Trivia Game can be played as individuals or in teams. Teams can be chosen in any manner.

Play: When the game starts the players start searching for Question and Answer Cards that match. Once a player finds a match that player holds on to the matching cards until the end of the game.

While searching for matching cards each player can hold and carry one unmatched Question Card. A player can set down an unmatched Question Card and pick up a new one at any time.

A player can pick up an Answer Card only long enough to read the answer to see if it matches the Question Card the player is holding. If the cards match the player keeps the cards, if they do not match the player must set the Answer Card down.

The game ends when all the cards have been picked up and matched. Each individual or team counts how many matches they have. The individual or team with the most matching cards wins.
